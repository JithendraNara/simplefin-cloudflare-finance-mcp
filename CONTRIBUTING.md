# Contributing

Contributions are welcome.

Before opening a pull request:

1. Run `npm run worker:typecheck`.
2. Run `npm run build`.
3. Keep examples generic and token-free.
4. Add or update docs for new MCP tools.
5. Do not include real finance data, D1 dumps, OAuth secrets, bearer tokens, or SimpleFIN Access URLs.

Good public examples use:

- `finance.example.com`
- `<MCP_BEARER_TOKEN>`
- `<ADMIN_TOKEN>`
- `your-github-login`
- fake UUIDs such as `00000000-0000-0000-0000-000000000000`
